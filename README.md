# easy-apply-job

A Chrome extension to assist with job applications on LinkedIn, Naukri, and MyWorkDay.

## Installation (Development Mode)

To run this extension locally:

1.  Open Chrome and navigate to `chrome://extensions/`.
2.  Toggle **Developer mode** (top right) to **On**.
3.  Click **Load unpacked**.
4.  Select the `c:\Extension\easy-apply-job\` directory.

## Reloading Changes
When you modify source files (like `content.js` or `content.css`), go to `chrome://extensions/` and click the **Reload** icon on the extension card to apply the updates.

## Building for Production

To package the extension for distribution:
1.  Zip all files in the project root (including `manifest.json`).
2.  Upload the `.zip` file to the Chrome Web Store Developer Dashboard.
